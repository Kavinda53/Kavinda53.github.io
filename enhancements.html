Enhancements Used

Index Page

  • In the index page, within the class the height of the page is set using 100vh which is useful for creating full height elements (100%) that fill up the entire viewport's height.
  • a navigation bar is styled using CSS setting the width for 95%.
  • Display flex is used to lay a collection of items out in one direction or another. margin is set to 0 to start the margin at the beginning of the right end of the web page.
  • The text-transform property is used to set the text in the navigation bar to capital.
  • Text decoration property which specifies the decoration added to the text is set to none to avoid decorations in the navigation bar.
  • The position property is set to absolute to set the navigation bar at the top. vertical position of a positioned element is set to -10px.
  • A transition property is set to 0.5s to make the red underlining procedure smooth and completed within 0.5 seconds.
  • Content'' property is used to insert generated content.
  • Transform: translate (-50) property is used to move content element left and upwards by 50% of the computed dimension along the axis
  • Line-height property is used to set the height to 25 pixels.
  • A button is created of width 200px. The cursor is set to pointer where the mouse changes to a pointer. Using overflow: hidden property, is used so that when the overflow is clipped, rest of the content is hidden
  • z-index controls the vertical stacking overlay of the webpage. Here -1 is used position the button behind other elements in the web page.
  • A transition is set to 0.5 seconds to make the hovering smooth.
  • The bottom 0 property is used to set the footer at the end of the web page.
  • The social media icons are defined in a class called .holder index and the height of the images are set to 35px.

Topics Page

  • Many enhancements added to the navigation bar of the index page are also applicable to the navigation bar in the topics page
  • A parallax effect is used to slide the web page content over an image used. The position of the image is set to relative. The background is set to no repeat to prevent repeating. The background is positioned to the centre.
  • The justify-content property sets the flexible navigation bar when the items do not use all available space on the main-axis horizontally.
  • The logo of the index page is positioned to absolute and the height of the image id set to 45px.
  • The display: inline-block is used to allow modifying the height and width of the navigation bar.
  • A .caption class is used to define the element that contains the "Explore Nest.JS" text. The position is set to absolute to prevent any alternations in its position. The text is aligned to the centre.
  • .caption .border is used to create the black background around the Explore Nest.JS topic.
  • Letter-spacing is used to add space between letters.
  • @media screen and (max-width:500px) is used to adjust the displaying of the navigation bar at different resolutions.

Quiz Page

  • The Nest logo was added using similar Enhancements used to add logos to the rest of the pages.
  • ENhancements made to the navigation bar is similar to the enhacnements made in rest rest of the web pages
  • A shadow type effect was added to the Quiz topic using the text-shadow property.
  • The background of the text inputs are made transparent, input[type=text]
  • The background of the radi0 buttons and checkboxes are transparent and the opacity is made to 0.6
  • An in-range input type is used and its background is set to transparent

About Us Page

  • a .wrpper class is defined to set the top margin to 5%.
  • Using .wrapper h1 a margin-bottom is set to 60 pixels.
  • A team class is defined to contain the the team member infromation.
  • Background color is set to white color setting margins of 20px to the bottom, and the inirtial width is set to 300px.
  • Position is set to relative.
  • .team .team_member p. role class sets the color to bright blue, sets a margin of 12 pixels on the top and bottom.
  • The text-transform is set to uppercase.
  • @media (max-width: 768px) applies to screen sizes smaller than or equal to 768px. It sets the width of `.team_member` to 100%
  • team_member:nth-child(-n+3) is a pseudo-class that is applied to the first three `.team_member` elements. It is set to -1, which changes their order on the page to be in reverse
