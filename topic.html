Explore Nest.JS

What is Nest.JS?

  1. In simple terms, nest.js is an open source node.js framework built using typescript
  2. for developing node.js server-side applications, which allows developers to cut loose complex configuration in backend development.
  3. Nest.JS makes uses HTTP Server Frameworks such as Express. In addition to providing an abstraction layer over Express and Fastify,
  4. Nest also makes their APIs available to developers directly. As a result, programmers are free to use any of the countless third-party modules that are accessible for the underlying platform.

Technology Of Nest.JS

Nest.js depends on angular architecture. Some core concepts used by nest.js which makes it unique.
Modules: - organisation of code, the code is broken down into logically and effective into reusable units.
Providers: - providers breaks down and elaborates any form of complexity and logic in a manner understandable by the developer.
Controllers:- coordinates incoming request and provides appropriate.

Nest can work with any Node HTTP framework after creating an adapter.
The two types of HTTP platfroms that are supported out-of-the-box are express and fantasy.

Node to Nest

It provides a collection of libraries and abstractions for developing server side applications in ease, both quickly and efficiently.
